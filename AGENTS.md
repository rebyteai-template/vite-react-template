# React + Vite Template

A minimal React 19 starter with Vite 7, TypeScript, and ESLint.

## Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite 7
- **Language**: TypeScript 5.9
- **Linting**: ESLint 9
- **Package Manager**: npm
- **Build Output**: `dist`
- **Dev Port**: 5173

## Available Skills

| Skill | Description |
|-------|-------------|
| `build-and-deploy` | Build and deploy to Vercel or Netlify |

## Project Structure

```
├── src/
│   ├── App.tsx       # Main App component
│   ├── main.tsx      # Entry point
│   └── App.css       # Styles
├── public/           # Static assets
├── index.html        # HTML template
└── vite.config.ts    # Vite configuration
```

## Quick Commands

```bash
# Install dependencies
npm install

# Development
npm run dev

# Build
npm run build

# Preview production build
npm run preview

# Lint
npm run lint
```

## Features

- React 19 with latest features
- Vite 7 with fast HMR
- TypeScript strict mode
- ESLint with React hooks rules
- React Refresh for fast development
