# Template Source

This template was created using Vite's official CLI:

```bash
pnpm create vite vite-react-template --template react-ts
```

## Original Source
- **Generator**: Vite CLI (https://vitejs.dev/)
- **Template**: `react-ts` (official Vite template)
- **Documentation**: https://vitejs.dev/guide/

## Customizations
- Added `netlify.toml` for Netlify deployment configuration
- Configured for Netlify Edge deployment with SPA routing

## Upstream Updates
To update this template with the latest Vite version:
```bash
pnpm create vite new-template --template react-ts
# Copy over customizations (netlify.toml, etc.)
```
